### tcp 协议
(1)开头：0x01;  DATA_HOT
(2)是否免费模式：0x01是免费模式;  0x02是收费模式  Free_Not
(3)是否 游戏打开中：0x01是打开播放器没有打开游戏，0x02是游戏打开中   In_Game
(4)游戏时间：0;3600 秒算发送   DATA_TIME
(5)是否链接成功：0x01是第一次连接成功唤醒，0x02没有连接成功    SK_CONNECTED
(6)结束码：0x11   END_CODE
(7)端口号：3217
(8)IP：192.168.188.xx
(9)数据格式{0x01;0x02;0x02;3000;0x01;0x99;}
